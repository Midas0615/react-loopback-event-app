# Event Application

### Setup:

- `yarn install` -- Server (terminal 1)
- `node .`
- `cd client && yarn install` -- Client (in terminal 2)
- `npm start`

Frontend runs on http://localhost:3000

API explorer: http://localhost.3000/explorer
